Il "tutor" è un'introduzione "pratica" per nuovi utenti dell'editor Vim.

La maggioranza dei nuovi utenti lo può completare in meno di un'ora.  Alla
fine, si sarà in grado di usare Vim per le modifiche più semplici a un file.

Il "tutor" è un file che contiene le lezioni introduttive. Dalla directory che
contiene il file che state leggendo si può semplicemente immettere "vim
tutor1.it" e poi seguire le istruzioni nelle lezioni.  Le lezioni spiegano
come modificare il file stesso, quindi NON VA USATA LA COPIA ORIGINALE del
file, ma occorre copiarlo prima da qualche altra parte, e poi usare il file
copiato.

Nei sistemi di tipo UNIX (e in Windows) basta eseguire lo script "vimtutor",
che genera per voi una copia modificabile del file che contiene le lezioni.

Quanto detto sopra riguardo al capitolo 1 del "tutor" vale anche per il
capitolo 2.  Però, per eseguirlo, va usato il comando "vim tutor2.it",
oppure "vimtutor -c 2".

Il capitolo 2 del "tutor" è stato scritto da Paul D. Parker.

Traduzioni
----------
I file tutor1.xx per il capitolo 1 e tutor2.xx per il capitolo 2 del
"tutor" sono delle traduzioni (dove "xx" indica il linguaggio).
La codifica di tutor1.xx e tutor2.xx dev'essere utf-8.
