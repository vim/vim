This is Vim's in-tree version of the Tree-sitter library. If you want to link
with your system version of Tree-sitter, add `--enable-treesitter=dynamic` to
configure.

Copied over from the main repo, but with all directory/files removed except the
bare minimum to build the C library + licenses.

The original can be found here: https://github.com/tree-sitter/tree-sitter
